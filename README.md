## ğŸš€ Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/amdadislam01/learn-hero-apps.git
   cd learn-hero-apps
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

3. Start development server:

   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser at the address (usually `http://localhost:5173`).
